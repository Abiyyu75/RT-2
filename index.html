Fitur Smooth Scrolling (`scroll-behavior: smooth;` di CSS) akan bekerja secara otomatis dengan `scroll-snap`. Namun, kita perlu **memperbaiki** *event listener* pada Smooth Scrolling agar mengarahkan ke bagian yang tepat: ```javascript // GANTI BAGIAN 3 DI script.js ANDA DENGAN INI // ========================================================== // 3. SMOOTH SCROLLING NAVIGASI // ========================================================== document.querySelectorAll('nav a[href^="#"]').forEach(anchor => { anchor.addEventListener('click', function (e) { e.preventDefault(); const targetId = this.getAttribute('href'); const targetElement = document.querySelector(targetId); if (targetElement) { // Gunakan scrollIntoView untuk memaksa browser menuju ke elemen // Dalam mode scroll-snap, ini akan memicu efek snapping yang mulus. targetElement.scrollIntoView({ behavior: 'smooth', block: 'start' // Pastikan elemen berada di awal viewport }); } }); });
